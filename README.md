# Image Classification using Support Vector Machine (SVM)

## Overview
This project demonstrates image classification using a Support Vector Machine (SVM). The model was trained, tested, and evaluated using image data.

## Objectives
- Load image dataset
- Preprocess images
- Train an SVM classifier
- Predict image classes
- Evaluate model performance

## Technologies Used
- Python
- NumPy
- Matplotlib
- Scikit-learn

## Machine Learning Algorithm
- Support Vector Machine (SVM)

## Evaluation Metrics
- Accuracy
- Classification Report
- Confusion Matrix

## Outcome
Successfully implemented an SVM-based image classification model and evaluated its performance using standard classification metrics.
